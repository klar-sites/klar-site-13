Rich text

Design notes for a focused mobile launch

Use the prose wrapper for article bodies, CMS content, changelogs, and longer marketing explanations. It keeps links, headings, lists, quotes, and inline code visually aligned with the rest of the system.

Why it works

A compact spacing rhythm lets readers scan quickly while still giving important ideas room to breathe.

  • Clear headings create strong hierarchy.
  • Muted body copy supports calm reading.
  • Primary links remain easy to find.
